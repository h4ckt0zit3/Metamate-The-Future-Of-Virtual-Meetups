Login & Identity Verification

MetaMates eliminates traditional passwords entirely. Users authenticate via biometric scanning (face ID or fingerprint) combined with a Decentralized Identifier (DID) stored on the Ethereum blockchain. A zero-knowledge proof validates your identity without ever transmitting personal data. Your cryptographic key is held only by your device ‚Äî not our servers.

03

Real-Time Emotion Detection

Our proprietary Facial Emotion Recognition (FER) module runs entirely in the browser via TensorFlow.js. It tracks 68 facial landmarks per frame at 60fps and classifies 7 primary emotions (Happy, Sad, Angry, Surprised, Disgusted, Fearful, Neutral) with 99.2% accuracy. The detected emotion is simultaneously mapped to the avatar's facial rig and used to adapt room lighting, ambient sound, and UI coloring.

Collaborate Securely

Everything inside a MetaMates session is protected by end-to-end encryption (E2EE) using the Signal Protocol. Screen shares, shared whiteboards, file transfers, and voice channels are all encrypted. Every event ‚Äî joins, leaves, file uploads, chat messages ‚Äî is logged as an immutable transaction on the blockchain, creating a tamper-proof audit trail that can be verified by all participants at any time.
